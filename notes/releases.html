Bagaimana cara install ROM? Berikut cara install dengan benar:

  • Siapkan file ROM dan Firmware yang telah diunduh sesuai perangkat
  • Masuk ke mode pemulihan seperti TWRP dan semacamnya dengan cara menekan tombol power dan volume atas secara bersamaan saat perangkat keadaan mati
  • Lakukan Wipe Dalvik/Cache, Cache, System, Vendor, Data
  • Flash firmware sesuai dengan versi Android yang akan diinstal, silahkan cek FIrmwarenya pada halaman tabel dibawah.
  • Flash ROM yang telah diunduh
  • Jika ROM versi Vanilla (ROM tanpa Google Apps), disarankan untuk instal Google Apps seperti OpenGApps, dan sebagainya. Namun lewatkan langkah ini jika ROM sudah termasuk Google Apps
  • (Opsional) Flash Disabe Force Encryption jika dibutuhkan
  • (Opsional) Flash Magisk jika ingin perangkat keadaan Root
  • Langkah terakhir, wajib lakukan Format Data kemudian Mulai Ulang ke Sistem

Jika langkah-langkah tersebut berhasil, maka perangkat Anda sudah terinstal Custom ROM dan siap digunakan dengan normal.

Firmware yang tersedia

  • Pastikan gunakan Firmware yang sesuai dengan versi Android pada ROM diatas
  • Jika Custom ROM berbasis Android 10, gunakan Firmware Android 10
  • Jika Custom ROM berbasis Android 11 atau keatas, gunakan Firmware Android 11
  • Instal atau flash Firmware dilakukan sebelum flash Custom ROM saat di mode pemulihan (TWRP)

Kernel yang tersedia

Hal yang perlu diperhatikan!

  • Jika terjadi bootloop atau stuck logo boot, artinya kernel tidak cocok dengan ROM.
  • Lakukan Backup boot dan dtbo kernel bawaan menggunakan TWRP untuk mencadangkan kernel bawaan jika mengalami bootloop setelah install kernel kustom
  • Untuk memulihkan kernel bawaan ROM yaitu masuk ke mode pemulihan (TWRP) dan pilih menu restore > restore boot dan dtbo, lakukan reboot

Langkah cara install:

  • Download Kernel yang akan digunakan
  • Masuk ke Mode Pemulihan (TWRP)
  • Lakukan Wipe Cache dan Dalvik
  • Ke menu Install > Flash Kernel yang telah didownload
  • Reboot ke Sistem
