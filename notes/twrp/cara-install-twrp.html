Cara Install Recovery Kustom TWRP

1. Hal Yang Harus Diperhatikan

  • "Do with your own risk!" penulis tidak bertanggung jawab atas kerusakan yang terjadi akibat proses rooting smartphone.
  • Backup data Anda terlebih dahulu.
  • Perangkat Android dalam kondisi Unlocked Bootloader.

2. File dan Environment yang harus disiapkan

  • Siapkan PC atau Laptop
  • Pastikan PC/Laptop terinstall Minimal ADB Fastboot
  • TWRP yang telah didownload

    • Download TWRP 3.7.0_12-0-ginkgo.img

  • Disable Force Encrypt
  • (Opsional) Magisk v25.2

3. Langkah untuk install TWRP

  • Masuk mode fastboot dengan menekan tombol Volume Down + Power dari keadaan mati, tahan selama 5 detik hingga muncul logo fastboot.
  • Jalankan Minimal ADB and Fastboot yang telah terinstal pada PC/Laptop dengan klik kanan run as administrator pada file cmd-here.exe. lalu sambungkan device via usb cable.
  • Flash Recovery: jalankan kode dibawah ini, (sesuaikan nama file twrp dengan yang ada, seperti nama twrp-3.7.0_12-0-ginkgo.img)
    fastboot devices
    lakukan flash
    fastboot flash recovery twrp-3.7.0_12-0-ginkgo.img
    setelah selesai masuk ke Mode Recovery dengan menekan dan tahan tombol power dan volume atas secara bersamaan hingga boot atau dengan cara ketik dibawah ini di ADB
    fastboot boot twrp-3.7.0_12-0-ginkgo.img
    Jika masuk recovery geser ke kanan swipe to allow modification dan ketuk tombol wipe -> format, ketik "yes" lalu confirm. proses ini akan menghapus semua data perangkat anda agar memori internal tidak terenkripsi. Pastikan jangan lupa backup data pribadi.
  • (Opsional) Flash Magisk untuk Root Perangkat, copy file magisk-Vxx.x.zip ke internal storage. lalu install pada file magisk-Vxx.x.zip, Kemudian Swipe to confirm flash . tunggu sebentar lalu selesai.
  • Flash Disable Force Encrypt
  • Reboot -> Reboot to system

Langkah - langkah install twrp berhasil
